1) Use git command to clone the project or download the project as a Zip file.
```bash
git clone https://github.com/tylau-dev/JavaPathENProject8.git
```

# Usage
1) Run the following command line to run the project
```bash
cd "./TourGuide/"
./gradlew run
```
